# 🚀 快速部署指南

## 立即部署到Vercel（3步完成）

### 第1步：推送到GitHub

双击运行 `推送到GitHub.bat` 文件，或手动执行：

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/NYCTEAM/REF.git
git branch -M main
git push -u origin main
```

### 第2步：导入到Vercel

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 选择 `NYCTEAM/REF` 仓库
5. 点击 "Import"

### 第3步：部署

1. 项目设置保持默认（Vercel会自动检测Next.js）
2. 点击 "Deploy"
3. 等待2-3分钟构建完成
4. 获得部署URL（例如：`https://ref-xxx.vercel.app`）

## ✅ 部署完成！

现在可以访问：
- 主页: `https://your-app.vercel.app`
- 后台: `https://your-app.vercel.app/admin`
- 统计: `https://your-app.vercel.app/stats`

## 🔧 常见问题

**Q: 部署失败怎么办？**
A: 检查以下内容：
- GitHub仓库是否公开
- package.json是否正确
- 依赖是否都已安装

**Q: 如何更新部署？**
A: 推送新代码到GitHub，Vercel会自动重新部署

**Q: 如何自定义域名？**
A: 在Vercel项目设置中添加自定义域名

**Q: 数据为什么丢失？**
A: Vercel是无服务器平台，SQLite数据不持久化。需要使用云数据库。
